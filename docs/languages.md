## Ardrainic (Romance / Old French)

*11 dialects — 215 unique names*

A refined, courtly language family inspired by Old French phonetics. Each dialect transforms base names through regional sound shifts.

**How "Thibeval" sounds across dialects:**
> Thibeval → Thibevard (Sittadellian) → Thibevael (Aullerican) → Thibevalen (Aloisoise) → Thibevel (Caprainois) → Thibeva (Pwasciards) → Thibevan (Serenish) → Thibewald (Berenguer) → Thibevat (Draquatin) → Thibevault (Chevallois) → Thibevel (Appavin)

### Ardrainic (default)

| Type | Examples |
|------|----------|
| Male | Thibeval, Galenor, Lancelin, Percival, Florentin, Davriel |
| Female | Belisore, Eilenore, Dianore, Alisande, Iselinde, Durande |
| Dynasty | de Galenval, de Thiberen, de Corvelin, de Briseval, de Lotheren, de Morvaille |

### Dialect Variants

| Dialect | Male Shift | Example |
|---------|-----------|---------|
| **Sittadellian** | -al → -ard | Thibevard, Galenort, Lancelard |
| **Aullerican** | -al → -ael | Thibevael, Galenoer, Lancelein |
| **Aloisoise** | -al → -alen | Thibevalen, Galenoren, Lanceliner |
| **Caprainois** | -al → -el | Thibevel, Galenorel, Lancelien |
| **Pwasciards** | truncated | Thibeva, Galeno, Lancel |
| **Serenish** | -al → -an | Thibevan, Lancolin, Percivan |
| **Berenguer** | -al → -ald | Thibewald, Galenort, Lancelein |
| **Draquatin** | -al → -at | Thibevat, Galenoun, Lancelet |
| **Chevallois** | -al → -ault | Thibevault, Galenourt, Lancelault |
| **Appavin** | -al → -el | Thibevel, Lancelet, Percivel |

> All Ardrainic dialects share base female names and dynasties.

## Moran (Dark Latin)

*5 dialects — 121 unique names*

The classical, ecclesiastical language of the Moran civilization. Dark, imperious naming with systematic evolution across time-period dialects.

**How "Aeternus" evolves through time:**
> Aeternus (Classical) → Aiternos (Early) → Eterno (Late) → Eterno (Novus) → Eterno (Vivum)

### Moran (default — Classical)

| Type | Examples |
|------|----------|
| Male | Aeternus, Caelinus, Dominor, Mortivex, Nexius, Sanctorius, Severox |
| Female | Aeterna, Caelina, Dominara, Mortiva, Nexia, Sanctoria |
| Dynasty | Mortemari, Nextorii, Aeternii, Caelarii, Ossivarii |

### Dialect Variants

| Dialect | Shift | Male Example | Female Example |
|---------|-------|-------------|----------------|
| **Early Moran** | archaic -os endings | Aiternos, Cailinos | Aiterna, Cailina |
| **Late Moran** | simplified vowels | Eterno, Celino | Eterna, Celina |
| **Novus Moran** | added -e on -or stems | Eterno, Dominore | Eterna, Celine |
| **Vivum Moran** | living vulgate form | Eterno, Bellicore | Eterna, Celina |

> All Moran dialects share base dynasties.

## Vulgar Moran (Italian)

*8 dialects — 131 unique names*

A vulgarized offshoot of classical Moran, with Italian-sounding cadences and regional diversity.

**How "Aurelvan" sounds across dialects:**
> Aurelvan → Aurelvanus (Rovina) → Aurelvarn (Koth) → Aurelv (Thanok) → Aurelvander (Fluviolander) → Aurelvann (Fioranelle) → Aurelvanc (Marcheur) → Aurelvatti (Rattolini)

### Vulgar Moran (default)

| Type | Examples |
|------|----------|
| Male | Aurelvan, Valoren, Casperian, Lucernor, Domavel, Silvoran |
| Female | Aurelara, Valorene, Luceriel, Silvarine, Celovina, Doravene |
| Dynasty | di Ventura, di Rovina, dal Kotharn, di Thanor, van Fluvel, di Fiorane |

### Dialect Variants

| Dialect | Shift | Male Example |
|---------|-------|-------------|
| **Rovina** | classical -us/-o endings | Aurelvanus, Casperiano, Silvoranus |
| **Koth** | heavy -rn/-lt consonants | Aurelvarn, Casperiarn, Domavelt |
| **Thanok** | truncated, blunt | Aurelv, Valor, Casperic |
| **Fluviolander** | -der/-an river-folk suffixes | Aurelvander, Valorender, Domavelan |
| **Fioranelle** | -nn/-rt refined endings | Aurelvann, Valorenn, Domaveld |
| **Marcheur** | -nc/-our march-lord endings | Aurelvanc, Valorenc, Domaveu |
| **Rattolini** | -tti rat-folk diminutives | Aurelvatti, Valoretti, Domavetti |

> Rattolini also has unique female names (Aurelatta, Valoretta...) and character overrides (see [Character Overrides](character-overrides.md)).

# Etra: Names & Cultures

A submod for [Etra: The New Age](https://forum.paradoxplaza.com/forum/threads/etra-the-new-age.1732025/) that replaces generic naming with **1,200+ unique fantasy names** across **7 languages and 34 dialects**, complete with custom ruler titles, polity names, and character overrides.

> Every culture sounds different. Every dialect has its own flavor.

## At a Glance

| Feature | Count |
|---------|-------|
| Languages | 7 |
| Dialects | 34 |
| Unique names | 1,200+ |
| Ruler title variants | 4 ranks x 34 dialects |
| Translated polity names | 7+ countries x 7 languages |
| Government types | Monarchy, Theocracy, Republic, Military Order |

## Character Overrides

Specific characters receive hardcoded name variants to match their dialect:

- **Pantagrin** (Sordrenic): Alric → Aldronc, Anna → Gronnda, Magnus → Magnortz
- **Rattolini** (Vulgar Moran): Gnavius → Gnavio, Vernacius → Vernaci, Cecile → Cecisca
- Includes dynasty overrides (Sunbreaker → Destrazol, Lastoath → Darnegramen)
- Includes country name overrides for rat-folk polities (Jessaros, Vicitz, Voserro...)

## Installation

Subscribe on the Steam Workshop, or copy this folder to your Victoria 3 mod directory:
- **Windows**: `%USERPROFILE%\Documents\Paradox Interactive\Victoria 3\mod\`
- **Linux**: `~/.local/share/Paradox Interactive/Victoria 3/mod/`

## Compatibility

- **Multiplayer**: Yes (synchronized)
- **Save compatible**: Yes (no replace_paths)
- **Load order**: After Etra: The New Age

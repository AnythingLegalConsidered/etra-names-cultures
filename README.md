# Etra: Names & Cultures

A submod for [Etra: The New Age](https://forum.paradoxplaza.com/forum/threads/etra-the-new-age.1732025/) that adds unique fantasy names and dialect variants for all Etra cultures.

## Features

- Unique name pools for each culture group (Ardrainic, Torrent, Moran, Hravevi, Sordrenic, Armonorican, Vulgar Moran)
- Dialect-based naming variants within each culture
- Custom ruler titles reflecting each culture's identity
- Character name overrides for lore-accurate naming

## Requirements

- [Victoria 3](https://store.steampowered.com/app/529340/Victoria_3/) (v1.1.*)
- [Etra: The New Age](https://steamcommunity.com/sharedfiles/filedetails/?id=3619896881) (v0.05+)

## Installation

Subscribe on the Steam Workshop, or copy this folder to your Victoria 3 mod directory:
- **Windows**: `%USERPROFILE%\Documents\Paradox Interactive\Victoria 3\mod\`
- **Linux**: `~/.local/share/Paradox Interactive/Victoria 3/mod/`

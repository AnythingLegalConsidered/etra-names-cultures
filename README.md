# Etra: Names & Cultures

A submod for [Etra: The New Age](https://forum.paradoxplaza.com/forum/threads/etra-the-new-age.1732025/) that gives every culture its own voice — **1,200+ unique fantasy names** across **7 languages and 34+ dialects**.

> Every culture sounds different. Every dialect has its own flavor.

## What It Does

Instead of generic English names, rulers, dynasties, and countries get names that match their culture's language. A single name shifts across dialects:

```
Sigwald (Heuhian) → Sigewaeld (Fullhorner) → Seegwoud (Naurmen) → Sigwasz (Ratfolk)
Thibeval (Ardrainic) → Thibeva (Pwasciards) → Thibevault (Chevallois)
Llewyn (Armonorican) → Levin (Ebiam)
```

Ruler titles, polity names, and even specific character names all follow the same logic.

## Game Settings

The mod adds a **game rule** to toggle fantasy naming:

| Setting | Effect |
|---------|--------|
| **English** (default) | Vanilla English names |
| **Fantasy Languages** | Culture-specific fantasy names |


## Installation

Subscribe on the Steam Workshop, or copy to your mod directory:
- **Windows**: `%USERPROFILE%\Documents\Paradox Interactive\Victoria 3\mod\`
- **Linux**: `~/.local/share/Paradox Interactive/Victoria 3/mod/`

**Load order**: After Etra: The New Age
**Multiplayer**: Yes | **Save compatible**: Yes
